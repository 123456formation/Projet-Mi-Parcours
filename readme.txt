Synchronisation de répertoires------------------------------
Projet de mi-parcors  : 12/04/2021  participant : Marouane , Alexandre et Thibault 

=========

Ce document intitulé « Synchronisation  » issu du projet mi-parcours , l'outil a été réaliser dans le but de revoir les notions traités 
lors de la formation Devops.

cet outil à été réaliser en s'appuyant sur les differents codes open source disponible sur le web.

Vous pouvez copier, modifier des copies de cette source.

Description :
=============

ce  programme en python < synchronisation>  est un programme qui permet  de syncroniser des dossiers de sauvegarde vers des dossiers de destinations.

l'outil il va creer aussi un fichier log pour avoir l'historique de tous les logs, comme ça en cas de bug , vous pouriez consulter vos 

logs afin de syncroniser seulment les fichiers qui n'ont pas été syncroniser.

la function pause presente dans l'application elle ne marche pas , elle est en  cours d'amélioration , si vous arrivez à l'ameliorer n'hesitez pas à paratager votre solution =)


Vous pouvez utiliser  l'outil soit en  ligne de commande soit avec  l'interface graphique (tkinter).


Pour l'interface graphique vous pouvez  executer le code via le bottton run python present dans les differents editeurs de textes ou vous pouvez creer un exe.
